- Initial release of Flazz
- Multi-provider LLM support (OpenAI, Anthropic, Google, Mistral, Groq, Ollama, OpenRouter, Azure)
- Autonomous learning and memory system with knowledge graph
- AI-generated skills that improve over time
- Composio integration for 100+ platform connections
- MCP (Model Context Protocol) server support
- Local-first architecture with privacy focus
- Real-time streaming responses
- File attachment support in chat
- Wiki-style note linking with [[brackets]]
- Full-text search across workspace
- Customizable UI themes

## Release Notes Format

Each release will include:

- **Added**: New features
- **Changed**: Changes to existing functionality
- **Deprecated**: Features that will be removed in future versions
- **Removed**: Features that have been removed
- **Fixed**: Bug fixes
- **Security**: Security vulnerability fixes
